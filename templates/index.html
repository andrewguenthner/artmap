  • Oakland Mural GeoData Updater (current)

Press the button above to update mural data. You will then see a message indicating the update was successful. Then press the download button below to save the data. If you hit the download button before updating, nothing will happen.

Note that on some browsers, a page with odd-looking text may open. If so, use the "Save as" function (try right-clicking or press and hold to get a context menu) and save the file.
